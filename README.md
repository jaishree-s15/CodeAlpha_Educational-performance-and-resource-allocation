# 📊 Student Performance & Educational Insights Dashboard

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** that analyzes student academic performance and identifies the factors influencing educational outcomes. The dashboard enables educators and administrators to monitor student performance, evaluate learning patterns, and make data-driven decisions to improve academic success.

The project was developed as part of the **CodeAlpha Power BI Internship – Task 4: Educational Performance and Resource Allocation**.

---

## 🎯 Objectives

- Analyze overall student academic performance.
- Identify factors affecting final grades.
- Evaluate the impact of study habits, attendance, and educational support.
- Discover performance trends across different student groups.
- Provide actionable insights to support educational decision-making.

---

## 📂 Dataset

**Dataset:** Student Performance Dataset (student-mat.csv)

The dataset includes information about:

- Student demographics
- Study time
- Previous failures
- School support
- Family background
- Internet access
- Absences
- Final grades (G1, G2, G3)

## 📈 Key Insights

- Students who dedicate more time to studying generally achieve higher final grades.
- Previous academic failures are associated with lower academic performance.
- Higher absenteeism tends to negatively impact student outcomes.
- Educational support and internet access can positively influence student performance.
- Student lifestyle factors also show relationships with academic achievement.

---

## 💡 Recommendations

- Provide additional academic support for students with previous failures.
- Monitor and reduce absenteeism through early intervention.
- Encourage effective study habits and time management.
- Improve access to educational resources and support programs.
- Use data-driven insights to identify students requiring personalized assistance.
